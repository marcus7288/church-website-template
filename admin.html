🚧 GitHub Token Setup

Paste your GitHub Personal Access Token in the green bar above to enable one-click publishing. Your token is saved only in your browser and sent only to the GitHub API. Steps to create one:

  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Click Generate new token
  3. Under Repository access choose Only select repositories → faithagspringfield
  4. Under Permissions → Repository permissions set Contents to Read and write
  5. Click Generate token, copy it, and paste it in the field above

🔒 Change Admin Password

Enter a new password to generate its hash, then replace STORED_HASH in admin.html on GitHub.
